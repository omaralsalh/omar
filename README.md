- [PartyKit](https://partykit.io)

## How to Use

pull the branch 


### Install dependencies

You can use one of them `npm`, `yarn`, `pnpm`, `bun`, Example using `npm`:

```bash
npm install
```
### Run partykit

```bash
npx partykit dev
```


### Run Next js

```bash
npm run dev
```
